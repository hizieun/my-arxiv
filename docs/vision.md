# Vision — my-arxiv

## 왜 만드는가

쏟아지는 AI 논문을 따라가기 위해 여러 사이트·트위터·뉴스레터를 오가는 게 번거롭다.
**하나의 피드 + 내 노트 시스템**으로 통합하고 싶다. arXiv·HuggingFace 큐레이션을 한 곳에 모으고, 내가 읽은·읽을 논문에 짧은 메모를 남겨 다음에 다시 만났을 때 컨텍스트가 살아있게 만든다.

## 누구를 위해

**1차 사용자: 본인 (jieun)** — AI/ML 분야에 관심있는 개발자, 본업 외 사이드로 논문 트래킹. 매일 30분~1시간 짧게 훑는 패턴.

**잠재 사용자 (현재 비목표):** 비슷한 패턴의 다른 개발자. 멀티유저 도입은 "언젠가는" 수준의 장기 후보.

## 핵심 가치 제안

1. **통합** — arXiv + HuggingFace를 한 피드로 합쳐 중복 제거
2. **내 흔적이 남음** — 읽음·노트·요약이 누적되어 컨텍스트가 살아남
3. **AI가 부담을 덜어줌** — abstract 한국어 3줄 요약으로 빠른 선별
4. **단순함** — 외부 의존성 최소, 본인 브라우저에서 완결 (localStorage)
